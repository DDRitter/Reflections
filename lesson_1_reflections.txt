Puedes llevar un seguimiento de todos los cambios en el código y puedes reutilizar código que hayas descartado en algún paso previo para otros programas.



What do you think are the pros and cons of manually choosing when to create a commit, like you do in Git, vs having versions automatically saved, like Google Docs does?

Elegir cuando grabar permite que cada commit sea compilable y tenga sentido como entidad individual.
Por otro lado, el hacer grabaciones periódicas automáticamente sirve como copia de seguridad en caso de olvidos.



Why do you think some version control systems, like Git, allow saving multiple files
in one commit, while others, like Google Docs, treat each file separately?

Git está diseñado específicamente para tratamiento de control de versiones,
mientras que Google Docs tiene un objetivo más general y sólo ofrece un historial de cambios individual.
